Gestures

General

swipe down: undo

swipe up: redo

swipe right/left: change mode

implode: reset

explode: exit AR

Place

double tap: place volume on cursor/hide volume

pan: rotate volume

pinch: scale volume

twist: roll volume

hold: move volume

Inspect

pan: rotate planes

twist: rotate all 3 planes

hold: move planes

hold center: move all 3 planes

double tap: hide/reveal plane

pinch: scale volume

Edit

pan: rotate volume

twist: change image contrast

hold: edit mask

double tap: change brush mode

pinch: scale volume
